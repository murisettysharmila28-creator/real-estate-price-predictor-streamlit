# Real Estate Price Prediction

## Project Overview
This project predicts real estate prices using machine learning models built in Python and deployed with Streamlit.

## Dataset
The dataset contains housing-related features such as property tax, insurance, square footage, lot size, and property type.

## Models Used
- Linear Regression
- Random Forest Regressor

[show folder structure]

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Train the model:
   python main.py
4. Run the Streamlit app:
   streamlit run app.py

## Features
- Modular code structure
- Logging and error handling
- Model training and evaluation
- Price prediction through Streamlit UI
